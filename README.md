# home-dashboard
its one of my first portfolio project A modern, responsive home automation dashboard for monitoring and controlling smart home devices with real-time energy consumption tracking and device management. design gotten from dribble i develop the project with the aim of perfecting my html and css  able to use bootstap efficently and also graphs

## Live @
https://osigbemhe2020.github.io/home-dashboard/

## technology used
* html
* css
* javascript
* bootstrap
* chart.js

## key feature
* Centralized Device Control: Manage all smart devices from a single interface
* Energy Monitoring: Track real-time and historical energy consumption
* Responsive Design: Optimized for both desktop and mobile devices

## Key Components
* Grid Layout: Uses CSS Grid for responsive layout management
* Progress Circle: Custom CSS-based circular progress indicator
* Chart Integration: Chart.js for energy consumption visualization
* Toggle Switches: Custom CSS toggle switches for device control
